RepositoryBeanFactoryPostProcessor

@Component
open class RepositoryBeanFactoryPostProcessor : BeanFactoryPostProcessor, ResourceLoaderAware

A BeanFactoryPostProcessor that scans the specified base packages for repository interfaces and registers them as beans in the bean factory. This allows repository interfaces to be autowired by the ORM framework.
