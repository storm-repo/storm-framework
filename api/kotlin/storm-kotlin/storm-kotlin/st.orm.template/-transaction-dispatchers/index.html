TransactionDispatchers

object TransactionDispatchers

Selects a default dispatcher that prefers Virtual Threads on Java 24+, and falls back to Dispatchers.IO on older JVMs (to avoid VT pinning on Java 21–23).

You can override with: -Dstorm.virtual_threads.enabled=true|false
