# Java API Reference (Javadoc)

The aggregated Javadoc covers all Java modules in the Storm framework:

- `storm-foundation` - Shared interfaces and annotations
- `storm-core` - Core ORM implementation
- `storm-java21` - Java 21+ API with String Templates
- `storm-spring` - Spring Framework integration
- `storm-spring-boot-starter` - Spring Boot auto-configuration
- `storm-metamodel-processor` - Annotation processor for compile-time metamodel
- `storm-jackson2` - Jackson 2 JSON support
- `storm-jackson3` - Jackson 3 JSON support
- `storm-test` - JUnit 5 testing utilities
- `storm-postgresql` - PostgreSQL dialect
- `storm-mysql` - MySQL dialect
- `storm-mariadb` - MariaDB dialect
- `storm-oracle` - Oracle dialect
- `storm-mssqlserver` - SQL Server dialect
