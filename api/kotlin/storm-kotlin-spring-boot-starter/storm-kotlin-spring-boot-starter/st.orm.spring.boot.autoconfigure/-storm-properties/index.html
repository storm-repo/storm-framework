StormProperties

@ConfigurationProperties(prefix = "storm")
open class StormProperties

Configuration properties for the Storm ORM framework.

These properties are bound from the storm.* namespace in application.yml or application.properties. The auto-configuration builds a st.orm.StormConfig from these properties and passes it to the ORMTemplate factory.

Example configuration:


storm:
  ansi-escaping: false
  update:
    default-mode: ENTITY
    dirty-check: INSTANCE
    max-shapes: 5
