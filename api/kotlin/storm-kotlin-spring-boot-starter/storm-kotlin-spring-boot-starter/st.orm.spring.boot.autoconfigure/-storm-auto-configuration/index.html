StormAutoConfiguration

@AutoConfiguration
@ConditionalOnClass(value = [ORMTemplate::class])
@ConditionalOnBean(value = [DataSource::class])
@EnableConfigurationProperties(value = [StormProperties::class])
open class StormAutoConfiguration

Auto-configuration for the Storm ORM framework.

Creates an ORMTemplate bean from the available DataSource if no ORMTemplate bean has been defined by the user. A StormConfig is built from the bound StormProperties and passed to the ORMTemplate factory.
