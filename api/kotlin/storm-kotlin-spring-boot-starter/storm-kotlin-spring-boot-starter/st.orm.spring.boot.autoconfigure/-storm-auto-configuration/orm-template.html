ormTemplate

@Bean
@ConditionalOnMissingBean(value = [ORMTemplate::class])
open fun ormTemplate(dataSource: DataSource, properties: StormProperties, entityCallbacks: List<EntityCallback<*>>): ORMTemplate

Creates an ORMTemplate bean using the provided DataSource and StormProperties.

A StormConfig is built from the bound properties. Fields not explicitly configured in application.yml fall back to system properties and then to built-in defaults.

This bean backs off if the user has already defined their own ORMTemplate bean.

Return

a new ORMTemplate instance.

Parameters

dataSource

the data source to use for database operations.

properties

the Storm configuration properties bound from storm.*.
